# Множественное наследование

### Содержание проекта:
В пакете `src/` находятся классы `Product` и `Category`, для которых определены следующие свойства:
1. Для класса `Product`
- название (name),
- описание (description),
- цена (price),
- количество в наличии (quantity).
2. Для класса `Category` 
- название (name),
- описание (description),
- список товаров категории (products).
Для класса `Product` созданы два класса наследников: «Смартфон» (Smartphone) и «Трава газонная» (LawnGrass).
Класс `Smartphone` расширен атрибутами: производительность (efficiency), модель (model), 
объем встроенной памяти (memory), цвет (color).
Класс `LawnGrass` расширен атрибутами: страна-производитель (country), срок прорастания 
(germination_period), цвет (color).
Для класса `Category` также добавлены два атрибута класса: количество категорий и количество товаров.
Для обоих классов определены магические методы `__str__`, `__add__`, `__iter__` и `__next__`.
Для всех классов реализованы ограничения сложения с помощью функций `issubclass()` и `isinstance()`.
 
Также в модуле `main.py` реализована дополнительная функция `load_categories_from_json`, которая подгружает
данные по категориям и товарам, а также создает объекты классов из JSON-файла в папке `data/`.
