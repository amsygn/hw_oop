## Домашняя работа 15.1
# Магические методы в классах

### Содержание проекта:
В модуле `main.py` созданы классы `Product` и `Category`, для которых определены следующие свойства:
1. Для класса `Product`
- название (name),
- описание (description),
- цена (price),
- количество в наличии (quantity).
2. Для класса `Category` 
- название (name),
- описание (description),
- список товаров категории (products).

Для класса `Category` добавлены два атрибута класса: количество категорий и количество товаров. 
Доступ к этим атрибутам есть у каждого объекта класса, и в них хранится общая информация для всех объектов.
Атрибуты класса заполняются автоматически при инициализации нового объекта.

Также в модуле `main.py` реализовано дополнительное задание: функция `load_categories_from_json` подгружает
данные по категориям и товарам, а также создает объекты классов из JSON-файла, который находится в папке `data/`. 

В домашнем задании 15.1 добавлены магические методы `__str__`, `__add__`. Также выполнено дополнительное задание - 
добавлены магические методы `__iter__` и `__next__`. 

В корне проекта находится модуль тестов, которые проверяют:
- корректность инициализации объектов класса `Category`,
- корректность инициализации объектов класса `Product`,
- подсчет количества продуктов,
- подсчет количества категорий,
- выполнение магических методов.

Отчета модуля тестирования находится в файле [coverage.md](coverage.md).
